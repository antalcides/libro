\section{Introducción}

A veces un video o animación permite explicar de mejor manera un comportamiento que simplemente usando imágenes. Pstricks permite dibujar en \LaTeX\ usando calidad vectorial y el package \textbf{animate} realiza una animación usando imágenes en formatos conocidos o también con las herramientas de Pstricks o Tikz. En este documento se explicará cómo incorporar una animación a un documento o presentación usando pstricks.

Antes de leer este documento se recomienda estudiar brevemente los comandos de Pstricks y la notación PostScript.

\section{Package animate}

Primero se debe cargar el package en el preámbulo del documento, usando el siguiente comando:

\begin{verbatim}
\usepackage[*opciones]{animate}
\end{verbatim}

Las \verb=*opciones= disponibles se describen a continuación y al ser definidas en el preámbulo afectarán todas las animaciones que se inserten en el documento. Se recominda usar estas opciones como parámetros en los comandos o entornos que se detallan en la sección. %\ref{sec:com}.

\begin{enumerate}
	\item \verb+poster=primero|ninguno|último+

Especifica el cuadro que aparecerá cuando la animación esté inactiva. Puede ser el primero, el último o ninguno. Por defecto aparece el primer cuadro o frame

	\item \verb+every=<num>+
	
Se construye la animación solamente desde cada \verb+<num>+ avo cuadro. Los cuadros saltados son descartados y no se incluyen en el documento.

\section{Comandos y entornos útiles}
%\label{sec:com}

El package animate tiene dos formas de crear una animación. 
\begin{verbatim}
\animategraphics[opciones]{fps}{nombre_base}{primero}{último}
\end{verbatim}

Existe un comando que combina varios archivos existentes tales como figuras o pdf's. En caso de usar este comando es necesario cargar el package \textbf{graphicx}. \verb=fps= indice el número de cuadros por segundo de la animación.

Todos los archivos de la secuencia deben estar en la carpeta del código fuente y estar numerados consecutivamente \verb=nombre_base= es el nombre común a todos los archivos. Por ejemplos si los archivos a incorporar son \textit{fotin2, fotin3, fotin4 y fotin5} el nombre común es \textbf{fotin}. \verb=primero= es el número del primer archivo (en este caso 2) y \verb=último= es el número del último (en este caso el 5). No es necesario incorporar la extensión del archivo, ya que el package realiza una búsqueda automática, comenzando por los archivos eps.

\textbf{Ejemplo}: Se han guardado las imágenes eps fotin1, fotin2, fotin3.

\begin{verbatim}
\begin{center}
\animategraphics[scale=1, controls, loop, final]{1}{fotin}{1}{3}
\end{center}
\end{verbatim}



A continuación se muestra el entorno que permite crear una animación a partir de lo que se dibuja o escribe en cierto espacio en el código fuente.

\begin{animateinline}[loop]{2}
... typeset material ...

\newframe[2]

Lol
\end{animateinline}

\section{Prueba}

\begin{center}
\animategraphics[scale=1, controls, loop, final]{1}{animate_}{0}{53}
\end{center}
Si tienes un archivo fotin.gif podemos usar el programa convert desde la linea de comando asi:
\begin{verbatim}
convert fotin.gif -coalesce animate_%d.pdf
\end{verbatim}

o
\begin{verbatim}
convert fotin.gif -coalesce animate_%d.eps
\end{verbatim}
